找不到 API Key？

用瀏覽器打開任意 Flickr 相簿頁面 → 右鍵「檢視網頁原始碼」(Ctrl+U) → 用 Ctrl+F 搜尋 flickr.api.site_key → 會看到類似 "api_key":"73d8a48f5c..." 的字串 → 複製那串值貼到上面。 這個 key 是 Flickr 網站自己前端在用的公開 key，每個訪客瀏覽頁面時 本來就會收到，不是任何人的私人憑證。

準備就緒。只能讀取「公開」相簿；分級內容（moderate）在未登入狀態下會被 Flickr 自動過濾，這是平台端的限制。
